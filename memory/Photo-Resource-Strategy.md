# Photo Resource Strategy for Portfolio Sites
## High-Quality Images Without AI Look

---

## Option 1: Free Stock Photo Sites (Recommended for Speed)

### Tier 1: Best Quality

**Unsplash** (unsplash.com)
- **License:** Unsplash License — free for commercial use, no attribution required
- **Strengths:** Curated, high-end photography, professional quality
- **Best for:** Hero images, featured article photos, lifestyle shots
- **Download:** High-res (up to 6000px), no signup required
- **Search tips:** Use specific terms: "man cooking," "gym workout," "office remote work"
- **Limitations:** Popular images appear everywhere

**Pexels** (pexels.com)
- **License:** Pexels License — free for commercial use, no attribution
- **Strengths:** Larger library than Unsplash, includes videos
- **Best for:** Supplementary images, textures, backgrounds
- **Download:** Multiple sizes available
- **Search tips:** Filter by orientation, color
- **Limitations:** Quality varies more than Unsplash

### Tier 2: Good for Specific Needs

**Pixabay** (pixabay.com)
- **License:** Pixabay License — free for commercial use
- **Strengths:** 6M+ images, includes illustrations and vectors
- **Best for:** Infographics, icons, technical diagrams
- **Limitations:** More generic quality, heavy on illustrations

**FoodiesFeed** (foodiesfeed.com)
- **License:** Free for commercial use with attribution
- **Strengths:** Food-specific, high quality
- **Best for:** FuelOptimal recipes
- **Limitations:** Requires attribution

**StockSnap** (stocksnap.io)
- **License:** CC0 — public domain
- **Strengths:** No attribution required
- **Best for:** General use
- **Limitations:** Smaller library

## Option 2: Generate with AI (Control Look)

### Midjourney V6 for Photorealistic Results

**The Secret:** Photographic language in prompts

**Formula:**
```
[Subject] + [Camera specs] + [Lighting] + [Film stock] + [Professional context]
```

**Examples:**

**For MaleOptimal (TRT/Health):**
```
Muscular man in his 40s, natural lighting, Canon EOS R5 with 85mm f/1.2 lens, shallow depth of field, documentary style, gym environment, shot on Kodak Portra 400, professional photography --ar 16:9 --style raw
```

**For FuelOptimal (Recipes):**
```
Overhead shot of carbonara pasta in white bowl, steam rising, natural window light, Canon 5D Mark IV with 50mm lens, f/2.8, food photography, shot on Fujifilm Provia, professional styling --ar 4:3 --style raw
```

**For RemotePivot (Remote Work):**
```
Man working on laptop in modern home office, natural daylight, Sony A7IV with 35mm f/1.4, candid documentary style, shallow depth of field, shot on Kodak Gold 200 --ar 16:9 --style raw
```

**Key Techniques:**
- Always include specific camera (Canon EOS R5, Sony A7IV)
- Always include lens (85mm f/1.2, 50mm f/1.8)
- Always include film stock (Kodak Portra, Fujifilm Provia)
- Add "documentary style" or "candid"
- Use "--style raw" flag (removes AI smoothing)
- Mention "professional photography" or "editorial"

### DALL-E 3 (If Midjourney unavailable)

**Prompt formula:**
```
Professional photograph of [subject], taken with Canon EOS R5, 85mm lens, f/1.8 aperture, natural window lighting, shallow depth of field, documentary photography style, high detail, realistic
```

**Best practices:**
- Request "photorealistic" explicitly
- Add "taken by professional photographer"
- Mention specific lighting conditions
- Include "high detail, realistic textures"

## Option 3: Hybrid Approach (Best Results)

**Step 1:** Generate base image with Midjourney
**Step 2:** Edit in Photoshop/GIMP for imperfections
**Step 3:** Add slight grain/noise to remove AI smoothness
**Step 4:** Adjust colors to film look (Portra, Fuji)

**Tools:**
- **Photopea** (free, browser-based Photoshop)
- **Canva** (add filters, grain overlays)
- **Remove.bg** (clean up backgrounds)

## Implementation Strategy

### Phase 1: Immediate (This Week)
- Download 10-15 hero images per site from Unsplash
- Add to existing articles as featured images
- Add homepage hero images

### Phase 2: Content Fill (Next 2 Weeks)
- Generate 5-10 AI images per site using Midjourney
- Focus on unique concepts (not available in stock)
- Edit to remove AI look

### Phase 3: Ongoing (Monthly)
- 2-3 new images per new article
- Seasonal updates (summer/winter themes)
- Test which perform best

## Technical Implementation

### Image Optimization
- **Format:** WebP (modern), JPEG fallback
- **Size:** Max 2000px wide for web
- **Compression:** 80% quality (balance)
- **Lazy loading:** Implement for performance
- **Alt text:** SEO + accessibility

### Storage
- **Current:** Git LFS or direct repo (slow)
- **Better:** Cloudinary, Cloudflare Images, or Vercel Edge
- **Best:** Image CDN with auto-optimization

### Attribution
- **Unsplash/Pexels:** Optional but appreciated
- **Midjourney:** None required
- **Hybrid:** Note AI + editing
