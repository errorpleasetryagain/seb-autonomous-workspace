# Amazon Affiliate Integration System
## Linking Strategy for Recipe Content

### Product Categories to Link

#### HIGH PRIORITY (Always Link)
| Category | Examples | Commission |
|----------|----------|------------|
| Kitchen Equipment | Pans, knives, appliances | 4-8% |
| Specialty Ingredients | Mirin, miso, ghee | 1-3% |
| Bulk Pantry | Rice cases, spice sets | 1-3% |
| Cookbooks | Recipe books | 4% |

#### MEDIUM PRIORITY (Sometimes Link)
| Category | Examples | When to Link |
|----------|----------|--------------|
| Canned Goods | Coconut milk, tomatoes | Bulk only |
| Spices | Individual jars | When in sets |
| Pasta | Premium brands | Quality matters |

#### NEVER LINK
| Category | Reason |
|----------|--------|
| Fresh produce | Amazon Fresh separate program |
| Single eggs | Not practical |
| Basic salt/pepper | Too generic |

### Link Placement Strategy

1. **Ingredient List** — Link specialty items with context
   ```
   - 200g De Cecco spaghetti [£2 on Amazon] — Rough texture holds sauce
   ```

2. **Equipment Section** — Link essential tools
   ```
   Equipment: Microplane grater [£12 on Amazon] — For fine cheese
   ```

3. **Deep Dive Sections** — Explain why product matters
   ```
   De Cecco uses bronze dies... [Buy on Amazon]
   ```

4. **Cost-Saving Tips** — Link bulk options
   ```
   Bulk buy saves 30%... [Case of 12 on Amazon]
   ```

5. **Shopping List** — Separate Amazon vs supermarket
   ```
   □ Spaghetti [Amazon]
   □ Chicken [Tesco/Sainsbury's]
   ```
