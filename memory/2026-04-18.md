## Lessons Learned

1. **IPv6 Binding Trap:** Next.js defaults to `::1` on macOS — always set `HOST=0.0.0.0` for local dev

2. **Subagent Limits:** Codebase >75KB causes timeouts in parallel spawns — use sequential or manual for large refactors

3. **GitHub Secret Scanning:** Aggressive — blocks pushes with any credential patterns in history, even if deleted

4. **OpenClaw CLI Integration:** 
   - `sessions_send` requires exact session key format (`agent:main:main`)
   - `sessions_spawn` returns sessionKey, not agentId
   - Always escape quotes in CLI commands

5. **Autonomy Config:** Timeout defaults are too short for complex tasks — set to 1 hour minimum
