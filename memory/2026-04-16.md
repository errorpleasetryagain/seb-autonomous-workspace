### Dyslexia Deck ✅ MVP COMPLETE
**Status:** Working app ready for use  
**Location:** `/Users/adamturton/.openclaw/workspace/projects/dyslexia-deck/`  

A markdown scratchpad purpose-built for Adam with dyslexia-friendly features:
- ✅ OpenDyslexic font toggle (fonts included)
- ✅ Adjustable line spacing (1.5x, 2.0x, 2.5x)
- ✅ Reading ruler / focus line (draggable overlay)
- ✅ Clean, minimal UI with dark mode
- ✅ File-based storage (~/Documents/DyslexiaDeck/notes/)
- ✅ Keyboard shortcuts (Cmd+N new, Cmd+O open, Cmd+P preview)
- ✅ Auto-save (debounced 500ms)
- ✅ Live markdown preview
- ✅ Word/character count

**Tech Stack:** Tauri v2 + React + TypeScript + CodeMirror 6

**Why this matters:** Existing tools are browser extensions (limited), bloated suites (overkill), or single-feature apps. Nothing combines clean markdown editing with dyslexia-friendly typography in one focused package.

**How to run:**
```bash
cd /Users/adamturton/.openclaw/workspace/projects/dyslexia-deck
npm install  # if not done
npm run tauri dev  # runs the app
```

**To package as native app:**
```bash
npm run tauri build
# Creates: src-tauri/target/release/bundle/macos/DyslexiaDeck.app
```
