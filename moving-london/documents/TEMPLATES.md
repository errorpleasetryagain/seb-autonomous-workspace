## 📧 Employer Reference Request Email

**Send to:** HR department or your line manager

```
Subject: Reference Request for Tenancy Application

Hi [Name],

I'm applying for a rental property in London and the letting agent requires an employer reference.

Could you please provide a reference confirming:
- My job title
- My start date
- My annual salary
- That I'm a permanent employee (or contract end date if applicable)

The reference can be emailed directly to [letting agent email] or provided to me as a signed letter on company letterhead.

I need this by [date] as I'm hoping to secure the property this week.

Thanks so much,
[Your Name]
[Your Phone Number]
```

## 📧 Previous Landlord Reference Request

**Send to:** Current/previous landlord or letting agent

```
Subject: Reference Request for New Tenancy

Hi [Name],

I'm applying for a new rental property and the letting agent has requested a landlord reference.

Could you please confirm:
- The address I rented
- The dates of my tenancy
- The monthly rent amount

[Your Name]
```

## 📋 Document Checklist (For Applications)

Keep copies of all these in Notion Documents database:

### For Each Tenant:
- [ ] **Passport or Driving Licence** (photo page)
- [ ] **Proof of Address** (bank statement, utility bill — last 3 months)
- [ ] **Last 3 Payslips**
- [ ] **Employment Contract** (optional, but helpful)
- [ ] **Employer Reference Letter** (see template above)
- [ ] **Previous Landlord Reference** (if applicable)
- [ ] **Right to Rent** (visa/BRP if not UK citizen)

### Joint Documents:
- [ ] **Combined Income Proof** (shows you meet 30x rent rule)
- [ ] **Guarantor Details** (if income < 30x annual rent)
  - Guarantor ID
  - Guarantor proof of income
  - Guarantor proof of address

## 💡 Tips for Applications

1. **Be first:** London rentals move fast. Apply within 24 hours of viewing if you love it.
2. **Have docs ready:** Keep a folder with all documents scanned and ready to email.
3. **Offer longer tenancy:** 18-24 months can make you more attractive than 12.
4. **No pets (initially):** If you have pets, don't mention until after application approved (legal grey area but common practice).
5. **Highlight stability:** Emphasise permanent jobs, long-term rental history, no children (if applicable — again, ethically dubious but common).
6. **Move fast on holding deposit:** If they offer it, pay immediately to take the property off the market.

## 🚚 Removals Company Questions

When getting quotes, ask:

- [ ] Is insurance included? What's the coverage?
- [ ] Do they pack, or just transport?
- [ ] Is there a fuel surcharge?
- [ ] What's the cancellation policy?
- [ ] Do they do same-day pickup/delivery?
- [ ] Any extra fees for stairs, no parking, etc.?
- [ ] Can they store items temporarily if needed?
- [ ] Payment terms (deposit, balance when?)

**Typical cost Bolton → London:** £600-1,200 depending on volume and service level.
